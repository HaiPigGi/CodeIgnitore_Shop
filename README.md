# Toko Sayurin
Toko  Sayurin merupakan aplikasi web toko online sederhana dengan demo produk penjualan sayur. Produk dan kategori dapat diubah dengan bebas (tidak hanya sayur), pengaturan (nama situs, rekening) dan sebagainya bisa diubah dengan bebas melalui dasbor.

##Fitur
- Login
- Register
- Keranjang belanja
- Checkout
- Sistem kupon Belanja
- Pembayaran melalui transfer bank
- Konfirmasi pembayaran
- Formulir kontak
- Testimoni

##Hak akses
- Admin
- Customer
